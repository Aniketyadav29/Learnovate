Java Program Structure

Every Java program must have a class with a main method, which is the entry point of the program.

public class HelloWorld {
    // Main method - program entry point
    public static void main(String[] args) {
        // Print statement
        System.out.println("Hello, World!");
    }
}

Key Syntax Rules

  • Java is case-sensitive: "MyVariable" and "myvariable" are different
  • Class names should start with an uppercase letter
  • Method names should start with a lowercase letter
  • Program file name must exactly match the class name (with .java extension)
  • Every statement must end with a semicolon (;)
  • Code blocks are enclosed in curly braces {}

Lifecycle of a Program

In Progress

Java Program Lifecycle Stages

  1. Writing Source Code: Creating .java files with Java code
  2. Compilation: Using javac to compile into bytecode (.class files)
  3. Loading: ClassLoader loads bytecode into memory
  4. Linking: Verification, preparation, and resolution of references
  5. Initialization: Static variables and initialization blocks executed
  6. Execution: JVM executes the bytecode
  7. Termination: Program exits either normally or due to error
